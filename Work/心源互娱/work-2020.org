** 整理新工程
*** 工程目录说明
+ Asset
  + ArtResources ==> 存放美术资源
  + Editor ==> 存放编辑器代码
  + LuaScripts ==> Lua代码
  + Plugins ==> 插件类的代码
    + ThirdPart ==> 第三方插件
  + Resources ==> 尽量不放东西，目前临时放了demo相关的资源，后面会删掉
  + ResourcesExt ==> 放正式资源，后面会打成AB
    + Scenes ==> 存放场景文件
    + UI ==> UI相关资源
      + Prefabs ==> UI相关Prefab
      + Atlas ==> UI图集
      + Texture ==> 单张图片
  + Scripts ==> C#代码
  + StreamingAssets
